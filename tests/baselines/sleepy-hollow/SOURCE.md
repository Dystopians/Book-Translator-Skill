# sleepy-hollow

- File: `sleepy-hollow.epub`
- Source: Project Gutenberg eBook #41, *The Legend of Sleepy Hollow*.
- Source page: <https://www.gutenberg.org/ebooks/41>
- Purpose: the smallest baseline for a practical conversion-to-publication
  smoke run. Repeated references to the Headless Horseman and related aliases
  also exercise long-range entity retrieval.

At the default target size of 6000 characters, a clean conversion should
produce roughly 21 chunks. This edition has little image coverage; use
`standard-alice` for image-path checks.
